The Piano Memory Game

Welcome to the piano memory game! Will you take on the challenge? You can play what you like now, but to start the game, you simply flip the switch to put the piano into game mode and click the start game button. You can flip back to free play mode at any time, by flipping the switch again. The game is simple, just follow the piano, just like the old simon game we all used to play. Good luck and have fun!
